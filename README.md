# NovaFlow

NovaFlow is a cross-platform Markdown note-taking application that uses AI to bridge recording and writing, organizing fragmented knowledge into readable notes.

## Core Features

- **Lightweight** - 25MB, free, no ads
- **Cross-platform** - Windows, macOS, Linux, Android, iOS
- **AI-powered** - Free AI and sync solutions with out-of-the-box RAG support
- **MCP integration** - AI tool integration support
- **Intelligent agents** - Automated note processing
- **Quick recording** - Capture fragmented information without disrupting workflow
- **Native Markdown** - Standard Markdown storage format

## Use Cases

**From Recording to Writing**

Traditional note-taking apps focus on writing, but NovaFlow makes it easier to record scattered knowledge points first.

NovaFlow is organized into three parts:

1. **Recording** - Quickly capture scattered thoughts and knowledge fragments without worrying about organization. AI will later structure and organize everything into coherent notes.

2. **AI Dialogue** - Interact with AI in real-time while recording to better understand and remember content.

3. **Notes** - Review and optimize your organized notes independently.

The workflow lets you focus on capturing information in the moment, while AI handles the heavy lifting of organizing and structuring your notes.

## Installation

Download the latest release for your platform from the [releases page](https://github.com/drewsephski/NovaFlow/releases).

### macOS Security Notice

When opening NovaFlow for the first time on macOS, you may see a security warning stating "Apple could not verify this app is free of malware." This happens because the app is not yet notarized by Apple (we're working on it!)

**To open the app:**
1. Right-click (or Control+click) the NovaFlow app
2. Select "Open" from the context menu
3. Click "Open" in the security dialog

*Alternative:* System Settings → Privacy & Security → click "Allow Anyway"
